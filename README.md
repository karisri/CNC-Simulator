A browser-based CNC machining simulator built using HTML, CSS, JavaScript, and Three.js. This project visually simulates CNC tool paths and material removal based on G-code instructions.

🚀 Features
Real-time 3D CNC machining simulation
G-code support (G0, G1 basic movement commands)
Step-by-step execution control
Run / Pause / Reset functionality
Back (undo step) support
Visual material removal effect
Tool size control (numeric input)
Material and removal color customization
Lightweight and runs directly in browser (no backend)
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Three.js (3D rendering)
📌 Use Case

Useful for:

Learning CNC programming
Visualizing G-code paths
Educational demonstrations
Basic CNC simulation experiments
